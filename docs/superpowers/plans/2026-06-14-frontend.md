# ChipScope 前端 Implementation Plan (Plan 4)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现 ChipScope 前端 SPA——K线图 + 筹码火焰图（横向）+ 指标面板 + 日期回放，对接 Plan 1-3 后端 API，让筹码分析可视化。

**Architecture:** Vite + React 18 + TypeScript SPA。ECharts 统一画 K线和筹码火焰图（审查确认替代 Lightweight-Charts，避免冗余）。Ant Design 做 UI 骨架（中文股票场景友好）。fetch + 自定义 typed API client 对接后端，React hooks 管理状态（不引入 Redux）。React Router 分 Dashboard / StockDetail。

**Tech Stack:** Vite 5、React 18、TypeScript、ECharts 5（+ echarts-for-react）、Ant Design 5、React Router 6、Vitest + @testing-library/react（纯逻辑测试）。

**关键设计决策：**
1. 图表统一 ECharts（K线用 candlestick series，火焰图用横向 bar + 颜色映射）。
2. API client 用 fetch + 泛型，类型与后端 schema 对齐（手写 TS 类型，不生成）。
3. 日期回放：本地状态 currentDate，拖动滑块重新查 `/chips?date=`，火焰图随之更新。
4. 开发时 Vite dev server 代理 `/api` 到 `http://localhost:8001`（后端 uvicorn），跨域由代理解决。
5. 前端测试聚焦纯逻辑（API client、数据转换、指标格式化）；组件视觉靠手动验证 + E2E 截图。

- [ ] **Step 2: api/client.ts（typed fetch）**

```typescript
const BASE = "/api";

export async function apiGet<T>(path: string, params?: Record<string, string>): Promise<T> {
  const url = new URL(BASE + path, window.location.origin);
  if (params) Object.entries(params).forEach(([k, v]) => url.searchParams.set(k, v));
  const resp = await fetch(url);
  if (!resp.ok) throw new Error(`API ${path}: ${resp.status}`);
  return resp.json() as Promise<T>;
}
```

- [ ] **Step 1: format.ts（MA 均线计算 + 数字格式）**

```typescript
export function ma(values: number[], period: number): (number | null)[] {
  const out: (number | null)[] = [];
  for (let i = 0; i < values.length; i++) {
    if (i < period - 1) { out.push(null); continue; }
    let s = 0;
    for (let j = i - period + 1; j <= i; j++) s += values[j];
    out.push(s / period);
  }
  return out;
}

export const pct = (v: number) => `${(v * 100).toFixed(2)}%`;
```

import ReactECharts from "echarts-for-react";
import type { KlineBar } from "../types/domain";
import { ma } from "../utils/format";

export default function KLineChart({ bars }: { bars: KlineBar[] }) {
  const dates = bars.map((b) => b.ts.slice(0, 10));
  const ohlc = bars.map((b) => [b.open, b.close, b.low, b.high]);
  const closes = bars.map((b) => b.close);
  const option = {
    xAxis: { type: "category", data: dates },
    yAxis: { type: "value", scale: true },
    legend: { data: ["日K", "MA5", "MA20"] },
    series: [
      { name: "日K", type: "candlestick", data: ohlc },
      { name: "MA5", type: "line", data: ma(closes, 5), smooth: true, lineStyle: { width: 1 } },
      { name: "MA20", type: "line", data: ma(closes, 20), smooth: true, lineStyle: { width: 1 } },
    ],
    tooltip: { trigger: "axis" },
  };
  return <ReactECharts option={option} style={{ height: 360 }} />;

## Task 4: 筹码火焰图组件（ECharts 横向）

**Files:** `src/components/ChipFlame.tsx`

横向 bar：Y 轴价格，X 轴筹码占比 ratio。颜色按 ratio 深浅映射（红多）。

- [ ] **实现**

```tsx
import ReactECharts from "echarts-for-react";
import type { ChipDistribution } from "../types/domain";

export default function ChipFlame({ chip }: { chip?: ChipDistribution }) {
  if (!chip) return <div>无筹码数据</div>;
  const entries = Object.entries(chip.distribution)
    .map(([price, ratio]) => ({ price: Number(price), ratio }))
    .sort((a, b) => a.price - b.price);
  const prices = entries.map((e) => e.price.toFixed(2));
  const ratios = entries.map((e) => e.ratio);
  const maxRatio = Math.max(...ratios, 1e-9);
  const option = {
    grid: { left: 50, right: 30 },
    xAxis: { type: "value", name: "占比", axisLabel: { formatter: (v: number) => `${(v * 100).toFixed(0)}%` } },
    yAxis: { type: "category", data: prices, inverse: true },
    series: [{
      type: "bar", data: ratios,
      itemStyle: {
        // 占比越高颜色越深（红）
        color: (p: any) => `rgba(220,40,40,${0.2 + 0.8 * (p.value / maxRatio)})`,
      },
    }],
    tooltip: { trigger: "axis", formatter: (xs: any) => `${xs[0].name}<br/>占比 ${(xs[0].value * 100).toFixed(2)}%` },
  };
  return (
    <div>
      <ReactECharts option={option} style={{ height: 400 }} />
    </div>
  );
}
```
